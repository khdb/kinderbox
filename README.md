kinderbox
=========

A Jukebox for kids using RFID to select mp3 album


* * *


GPIO Usages
===========


UI Buttons:
-----------


__GPIO <==> Buttons__

* 23   -> Play/Pause (toggle)
* 22   -> Volume down
* 4    -> Volume up
* 18   -> Next
* 17   -> Prev


16x2 LCD:
-----------

__GPIO <==> LCD__
* 24   -> EN
* 25   -> RS
* 2    -> D4
* 3    -> D5
* 27   -> D6
* 7    -> D7
